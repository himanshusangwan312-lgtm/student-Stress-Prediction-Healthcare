About Project

This system predicts student stress using Machine Learning and provides intelligent lifestyle and health-based guidance.

📌 Project Overview

The Student Stress Prediction System is an intelligent web-based application designed to analyze and predict the stress level of students using Machine Learning techniques. In today’s fast-paced academic environment, students face high levels of pressure due to exams, career uncertainty, and personal challenges. This system provides early detection of stress and suggests practical improvements.

🔥 Upgrade: This system now uses real-world lifestyle data instead of academic-only factors, making predictions more realistic and accurate.

⚙️ How the System Works

The application collects important behavioral and lifestyle inputs such as:

  • 👤 Age
  • 😴 Sleep Duration
  • 📱 Screen Time
  • 🏃 Physical Activity
  • 🧑‍🤝‍🧑 Social Interaction Level
  • 💼 Work / Study Hours

These inputs are processed using a trained Machine Learning model which predicts stress levels as Low, Medium, or High.

🚀 Key Features

  • 🤖 AI-based stress prediction system
  • 💬 Smart chatbot for mental support
  • 🏥 Doctor & hospital finder with ratings
  • 📊 Admin dashboard with analytics
  • 📍 Location-based doctor search
  • 🚨 Emergency support page
  • 📈 Real-time stress visualization (charts & progress)

💡 Benefits for Students

  • ✔ Early stress detection
  • ✔ Improved mental health awareness
  • ✔ Access to professional doctors
  • ✔ Personalized lifestyle suggestions
  • ✔ Easy and user-friendly interface

🧠 Technologies Used

  • 🐍 Python (Flask Framework)
  • 📊 Machine Learning (Scikit-learn - Random Forest)
  • 🗄 SQLite Database
  • 📈 Pandas & NumPy
  • 🎨 HTML, CSS, Bootstrap (Modern UI)

🎯 Project Objective

The main objective of this project is to create a system that helps students identify their stress levels early and take appropriate actions to improve their mental health. It also connects users with doctors and provides useful resources for academic and personal growth.
